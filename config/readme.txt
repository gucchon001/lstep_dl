# LINE Data Downloader - 設定ガイド

## 1. 設定ファイル (settings.ini)
- settings.ini.template をコピーしてsettings.iniを作成
- LINE関連のURLを設定
- スプレッドシートIDと各シート名を設定
- タグ設定を必要に応じて変更
- 待機時間の調整（必要な場合のみ）

## 2. 環境変数 (secrets.env)
- secrets.env.template をコピーしてsecrets.envを作成
- LINE_EMAIL: LINEログイン用メールアドレス
- LINE_PASSWORD: LINEログインパスワード
- GOOGLE_CREDENTIALS_PATH: Google APIクレデンシャルファイルのパス

## 3. よくあるエラーと対処方法
1. ログインエラー
   - メールアドレスとパスワードが正しいか確認
   - 二段階認証が有効な場合は無効化

2. スプレッドシートエラー
   - スプレッドシートIDが正しいか確認
   - クレデンシャルファイルのパスが正しいか確認
   - シート名が正しいか確認

3. ダウンロードエラー
   - ネットワーク接続を確認
   - ダウンロードフォルダのパスが正しいか確認
   - 十分な待機時間が設定されているか確認

4. セレクターエラー
   - selectors.csvの内容が最新か確認
   - ページ構造の変更がないか確認

## 4. サポート
問題が解決しない場合は、以下の情報を添えて開発者に連絡：
- エラーメッセージ
- 実行ログ
- 設定ファイルの内容（パスワード等の機密情報は除く）
